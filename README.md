# Slitch

## Установка
### Требования:
- веб сервер
- php

### Веб сервер
Нужно настроить перенаправление на `index.php`, 
и отменить перенаправления в определённых случаях в папке `public`.
Рабочий пример конфигурации:

    server {
        listen 80;
        listen [::]:80;

        root /home/danik/PhpstormProjects/Slitch-main/public/;
    
        index index.php;
    
        server_name slitch;
    
        gzip on;
    
        client_max_body_size 2m;
    
        location ~ ^/css/.+\.css$ {
            try_files $uri $uri/ =404;
        }
    
        location ~ ^/img/.+\..+ {
            try_files $uri $uri/ =404;
        }
    
        location ~ ^/js/.+\.js$ {
            try_files $uri $uri/ =404;
        }
    
        location ~ ^/files/.+\..+ {
            try_files $uri $uri/ =404;
        }
    
        location / {
            rewrite ^(.+)$ /index.php?url=$1;
            try_files $uri $uri/ =404;
        }
    
    
        location ~ \.php$ {
            include snippets/fastcgi-php.conf;
            fastcgi_pass unix:/var/run/php/php7.4-fpm.sock;
            fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name;
            fastcgi_param QUERY_STRING    $query_string;
        }
    }

### Composer
Установить всё из `composer.json`.

### Database
Исполнить скрипт `production_slitch_2021-04-15-2020.sql` в консоли к своей базе данных.

## Цель
Первоначальная цель - это изучить английский язык, так чтобы я мог: 
- составлять предложения правильно 
- выражать предложения правильно письменно и устно
- определять на слух и визуально, какие слова в материале
- понимать значения воспринятых на слух и визуально слов из материала

## Опыт
По моим наблюдениям я решил, что слово и его значение запоминается на дольше, когда:
- дольше перерыв между моментами, когда ты вспоминаешь значение слова восприняв его на аглийском языке
- более реалистичней представляешь значение слова
- больше чувств задействуешь в описании его значения

Чувства:
- тактильное
- слуховое
- визуальное
- вкусовое
- обонятельное
- остальные

## Польза проекта
Проект позволяет чаще использовать первый пункт из ___Опыт___.
Он может увеличить скорость нахождения слов, которые мне не понятны, в тексте.
Но будет немного ошибок в нахождении не понятных слов. Количество ошибок зависит от того на сколько хорошо понятны слова, имеющиеся в базе данных.

## Использование
Часть описана в диаграммах.

### Раздел "handle" 
Когда вы нажимаете `handle` для определённого материала, 
то открывается окно с основными возможностями приложения.
Возможности и как их реализовать:

#### 1. Сохранить слова в базу данных, как изученные или не переводимые:
- Отметить слова (learned/untranslatable)
- Нажать красную кнопку `save` в нижней навигационной панели

#### 2. Скопировать переводы слов:
- Нажать кнопку `Sentences` в нижней навигационной панели
- Активировать функцию браузера "перевести страницу"
- Пролистать все предложения до конца
- Нажать голубую кнопку

#### 3. Деактивировать все кнопки learned/untranslatable
- Нажать на красную кнопку `reset` в нижней навигационной панели

#### 4. Скопировать слова в изначальном
- Нажать на `Words` в нижней навигационной панели

### Полный цикл обработки материала
- Зарегистрируйтесь (если нет аккаунта)
- Войдите в аккаунт
- Нажмите `Add stuff` в верхней панели навигации
- Заполните данные
- Нажмите на кнопку `Add stuff`
- Нажмите на `handle` для добавленного материала
- Выполните первый пункт из "Раздел handle"
- Обновите страницу `F5`
- Выполните четвёртый пункт из "Раздел handle"
- Вставьте слова в таблицу
- Выполните второй пункт из "Раздел handle"
- Вставьте слова в ту же таблицу в соседнюю колонку
- Поместите страницу браузера и таблицу на экране
- Проверяйте и корректируйте переводы в таблице, подставляя значения в предложения.
  Прочитайте все слова, не пропускайте.
- Можете "отсортировать слова" 
- Через "определённое время" открывайте материал и понимайте его используя словарик 
- Через "определённое время" повторяйте предыдущий пункт

#### Определённое время
Я не знаю сколько это в идеале, да и у меня не получается всегда 
одинаково проходить весь цикл, по этому и идеальное время колеблется от случая 
к случаю. Вам надо самим пробовать и определить своё время, но главная идея здесь -
чем позже, тем лучше, но при этом надо, как можно больше значений слов вспомнить без 
словарика. 

#### Отсортировать слова
Если вы хотите распечатать на листах бумаги словарик или просто удобнее распределить 
слова в таблице, то вам может помочь функция `Sort words` в нижней навигационной 
панели в разделе `handle` для определённого материала. Функция позволит быстро
распределить слова в таблице из колонки в несколько. Пример:

| words          |
| -------------- |
| first          |
| second         |
| third          |
| fourth         |
| fifth          |
| sixth          |
| seventh        |
| eighth         |
| ninth          |
| tenth          |

Я хочу распределить на таблицу из табличек из двух колонок и трёх строк, чтобы
можно было не прокручивая таблицу прочитать несколько колонок. Я введу в окне 
функции `Sort words` в поле "Column count" "2" и в поле "Words count in column" "3". 
Нажму кнопку `Sort`. Скопирую каждую колонку и вставлю её в таблицу по очереди и
получиться так:

| words          | words          |
| -------------- | -------------- |
| first          | fourth         |
| second         | fifth          |
| third          | sixth          |
| seventh        | tenth          |
| eighth         |
| ninth          |

Так можно сделать с каждой колонкой словаря и из этого:

| Original words | Translated |
| -------------- | ---------- |
| first          | первый     |
| second         | второй     |
| third          | третий     |
| fourth         | четвёртый  |
| fifth          | пятый      |
| sixth          | шестой     |
| seventh        | седьмой    |
| eighth         | восьмой    |
| ninth          | девятый    |
| tenth          | десятый    |

Получиться:

| Original words | Translated | Original words | Translated |
| -------------- | ---------- | -------------- | ---------- |
| first          | первый     | fourth         | четвёртый  |
| second         | второй     | fifth          | пятый      |
| third          | третий     | sixth          | шестой     |
| seventh        | седьмой    | tenth          | десятый    |
| eighth         | восьмой    |
| ninth          | девятый    |

Это пригодиться при печати словаря, для того чтобы словарь занял весь лист, 
а не только две колонки.
